# Script dependencies

Perl version 5.10 or higher

Perl Modules:
* Params::Validate
* Readonly
* XML::LibXML

# Execs and bundles available within repository (generated based on the main script)

Some exec and bundles files are available here: [execs](execs).
The exec files and bundles are autogenerated based on the main script [src/scripts/smart_xml_analyzer.pl](src/scripts/smart_xml_analyzer.pl).

Windows exe file:

```aidl
execs\smart_xml_analyzer.exe example\sample-0-origin.html example\sample-1-evil-gemini.html make-everything-ok-button
```
